# Airbnb 시장 분석 리포트 설계 (Phase 1)

## 목적
지역 + 침실/욕실 조건 입력 → Airbnb 시장 데이터 수집 → 손님용(클라이언트 프레젠테이션) + 내부용(전략 분석) 두 Excel 리포트 자동 생성

## CLI 인터페이스

```bash
python market_report.py <지역> [--beds N] [--baths N] [--mode both|client|internal]
```

예시:
```bash
python market_report.py 충신동 --beds 3 --baths 2
python market_report.py 홍대  # 전체 시장 분석
```

출력:
- `market_report_충신동_2026-06-17_손님용.xlsx`
- `market_report_충신동_2026-06-17_내부용.xlsx`

## 데이터 수집 전략

### 날짜창 (2개)
- 평일창: 3주 후 월요일 → 화요일 (1박)
- 주말창: 같은 주 금요일 → 토요일 (1박)
- 각 창: B모드 크롤링 (22컬럼 상세)

### 머지 전략
- 기준: URL (숙소 고유 식별자)
- 결과: 숙소별 `price_weekday` / `price_weekend` 보유
- 주말 프리미엄: 양쪽 창에 모두 존재하는 숙소만으로 계산

## 상업용 vs 개인 분류 로직

### 스코어링 (2점 이상 = 상업용)
| 신호 | 점수 |
|------|------|
| property_type에 호텔/게스트하우스/호스텔 포함 | +3 |
| host_name에 상업 키워드 포함 | +2 |
| 소개글에 프론트/리셉션/데스크 키워드 2개+ | +2 |
| 숙소명에 호텔/게스트하우스 포함 | +1 |

### 상업 키워드
- property_type: 호텔, hotel, 게스트하우스, guesthouse, 호스텔, hostel, bed and breakfast
- host_name: 호텔, 게스트하우스, 호스텔, 스테이, 레지던스, 리조트, inn, stay
- 소개글: 프론트, 리셉션, 체크인 카운터, 24시간 운영, 프론트 데스크

## 가격 통계 방법론

### 이상치 제거
- IQR 방법: Q1 - 1.5×IQR 미만 또는 Q3 + 1.5×IQR 초과 = 이상치
- 클린 데이터셋으로 KPI 산출, 이상치는 내부용에서 별도 표시

### 핵심 통계
- P10, P25, P50(중앙값), P75, P90
- 평균, 표준편차
- 주말 프리미엄 %

### 추천 객단가 3단계
| 단계 | 기준 | 설명 |
|------|------|------|
| 스타터 | P40 | 리뷰 확보 초기 전략 |
| 표준 | P55 | 리뷰 10개 이상 기준 적정가 |
| 프리미엄 | P70 | 슈퍼호스트 등급 목표가 |
- 주말 프리미엄 자동 적용하여 평일/주말 각각 표시

## 손님용 리포트 구조 (4 시트)

### Sheet 1: 표지
- 배너: 리포트 제목 + 지역 + 분석일
- KPI 박스 3개: 수집 숙소 수 / 평균 1박가 / 평균 평점
- 서브 KPI: 개인 숙소 수, 상업용 숙소 수, 주말 프리미엄, 중앙값 1박가

### Sheet 2: 시장 분석
- 가격 구간별 분포 (데이터바 조건부 서식): ~3만 / 3~5만 / 5~8만 / 8~12만 / 12~20만 / 20만+
- 개인 vs 상업용 각각 카운트
- 평일 vs 주말 평균가 비교 테이블
- 침실 수별 평균가 테이블 (beds filter 없을 때)

### Sheet 3: 경쟁 현황
- Top 30 숙소 (평점 기준 내림차순)
- 컬럼: 순위, 숙소명, 구분(개인/상업), 침실, 욕실, 평점, 평일가, 주말가, 슈퍼호스트
- 상위 5개: 골드 배경 강조
- 평점 4.8+ : 초록, 4.5-4.8: 노랑 조건부 서식

### Sheet 4: 추천 객단가
- 3단계 추천 테이블 (평일/주말 각각)
- 근거: 분석 대상 숙소 수, 이상치 제거 후 클린 샘플 수, 데이터 수집일
- 주의사항: Airbnb 정책 변동, 계절 요인

## 내부용 리포트 구조 (5 시트)

### Sheet 1: 대시보드
- 전체 KPI (이상치 포함/제외 비교)
- 세그먼트별 breakdown (개인/상업)
- 이상치 목록 요약

### Sheet 2: 원본 데이터
- 기존 22컬럼 + 분류(개인/상업) + price_weekday + price_weekend + 이상치_플래그
- 자동필터, 헤더 고정

### Sheet 3: 세그먼트 분석
- 개인 vs 상업용 교차 분석표
- 침실별 × 욕실별 교차표
- 슈퍼호스트 vs 일반 가격 비교

### Sheet 4: 가격 분포 분석
- 퍼센타일 전체 테이블 (P10~P90)
- 이상치 목록 (URL 포함)
- 주말 프리미엄 개별 숙소별 수치

### Sheet 5: 수집 메타데이터
- 수집 일시, 날짜창 정보
- 지오코딩 결과 (위도/경도)
- 창별 샘플 수, 공통 숙소 수
- 분류 로직 버전 명시
