# Airbnb 시장 분석 개선판

이 폴더는 원본 프로젝트를 보존한 상태에서 따로 만든 개선판입니다. 원본 폴더의 파일은 수정하지 않고, 이 폴더 안의 파일만 개선했습니다.

## 실행

더블클릭:

```bat
Airbnb시장분석.bat
```

직접 실행:

```bat
py -3 gui_app.py
```

## 주요 개선점

- GUI 글자 크기, 보조 글자색, 로그창 색상, 날짜 입력칸 색상을 정리했습니다.
- 버튼과 로그, Excel 시트명에서 깨질 수 있는 이모지성 문자를 제거했습니다.
- M 모드에서는 체크아웃 입력을 비활성화해 기준 날짜만 쓰는 구조를 분명히 했습니다.
- 숫자 입력 검증을 강화했습니다. 예약률은 `하 <= 기준 <= 상` 순서와 0~1 범위를 확인합니다.
- A/B/M 실행 함수가 실제 생성된 결과 폴더를 반환합니다. 긴 작업 뒤에도 GUI가 정확한 결과 폴더를 엽니다.
- `requirements.txt`에 GUI 실행에 필요한 패키지를 추가했습니다.
- 배치 파일은 기존 Python 경로가 있으면 사용하고, 없으면 `py -3`로 실행합니다.
- 가격 추천 로직을 단순 P40/P55/P70 고정 방식에서 조정 방식으로 바꿨습니다.

## 가격 추천 로직

추천가는 세 단계입니다.

- 스타터: 처음 리뷰를 모으기 위한 낮은 가격
- 표준: 가장 먼저 써볼 기준 가격
- 프리미엄: 후기와 사진이 충분할 때 목표할 수 있는 높은 가격

가격을 정할 때 다음 항목을 같이 봅니다.

- 시장 가격 분포
- 평균 리뷰 수
- 주말 가격 차이
- 경쟁 숙소 평균 평점
- 비교 숙소 수
- 청소비 부담

예를 들어 경쟁 숙소 평점이 높거나 비교 숙소 수가 적으면 가격을 조금 낮춥니다. 리뷰가 많고 주말 가격이 강하면 가격을 조금 올립니다. 청소비가 높으면 손님이 보는 총액이 비싸지므로 1박가를 조금 낮춥니다.
